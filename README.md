# Fluent Python 학습 저장소

[Fluent Python: Clear, Concise, and Effective Programming](https://www.oreilly.com/library/view/fluent-python-2nd/9781492056348/) 책을 학습하며 정리한 예제 코드 모음입니다.

## 목적

- 책의 예제 코드를 직접 작성하고 실행해보며 고급 파이썬 개념 익히기
- 각 코드에 대한 이해 내용을 **docstring**으로 상세히 기록
- 나중에 다시 참고할 수 있는 개인 학습 아카이브 구축

## 내용

- 각 파일에는 코드와 함께 해당 개념에 대한 설명이 docstring으로 포함되어 있습니다.
